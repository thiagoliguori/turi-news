Configuração: chave da API Anthropic

Os resumos são gerados pelo Claude, direto do seu navegador. Cole aqui sua chave da API Anthropic. Ela fica salva apenas neste navegador (localStorage) e não é enviada a nenhum outro lugar. Para criar uma chave, acesse console.anthropic.com.
